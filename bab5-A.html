Sub Bab A: Komponen Sistem Komputer

Memahami komponen utama sistem komputer: perangkat keras, perangkat lunak, dan peran pengguna dalam mengoperasikan komputer.

Pengantar: Sistem Komputer

Sebuah sistem komputer terdiri dari tiga komponen utama, yaitu perangkat keras (hardware), perangkat lunak (software), dan pengguna (brainware). Cara kerja sistem komputer ini dapat diibaratkan seperti sistem pencernaan pada tubuh manusia.

Misalnya, saat merasa lapar, manusia mengonsumsi makanan, proses ini mirip dengan pemasukan data ke dalam komputer. Setelah masuk melalui mulut, makanan bergerak melalui kerongkongan dan kemudian diolah di lambung, tempat nutrisi seperti protein dan karbohidrat dipisahkan dan dipecah menjadi zat-zat yang berguna bagi tubuh. Begitu pula dengan komputer: setelah data dimasukkan, data tersebut diproses oleh perangkat pemrosesan.

Selanjutnya, sisa makanan yang tidak dapat dicerna akan dikeluarkan dari tubuh sebagai limbah. Dalam sistem komputer, data yang telah diproses akan disimpan terlebih dahulu di perangkat penyimpanan. Setelah itu, hasil akhir berupa informasi dikirim ke perangkat keluaran untuk ditampilkan atau digunakan. Alur kerja ini ditunjukkan pada Gambar di bawah

Analogi Sistem Komputer dengan Sistem Pencernaan

Makanan = Data Input → Mulut = Input Device → Lambung = Processing Device → Nutrisi diserap = Data diproses → Sisa dikeluarkan = Output ditampilkan

1. Perangkat Keras (Hardware)

Perangkat keras merupakan peralatan fisik dari sebuah komputer yang dapat disentuh dan dipindahkan. Perangkat keras sistem komputer dapat diklasifikasikan dalam empat kelompok yaitu: Input Device, Output Device, Processing Device, dan Storage Device. Unit-unit ini terhubung satu dengan yang lainnya.

Perangkat Input (Input Device)

Keyboard

Keyboard adalah papan ketik yang terdiri dari banyak tombol seperti huruf, angka, dan simbol. Digunakan untuk mengetik teks dan memberikan perintah ke komputer. Saat kita menekan salah satu tombolnya, komputer akan memprosesnya menjadi huruf yang muncul di layar atau menjalankan perintah tertentu yang kita inginkan.

Papan ketik untuk memasukkan data teks

Mouse

Mouse adalah alat penunjuk yang digunakan untuk menggerakkan kursor (tanda panah) di layar komputer. Dengan cara digeser di atas permukaan datar, alat ini membantu kita untuk memilih, membuka, dan mengatur file atau aplikasi dengan cara diklik.

Alat penunjuk untuk menggerakkan kursor

Scanner

Scanner adalah alat untuk membaca sebuah dokumen baik gambar maupun teks yang dapat disimpan secara digital ke dalam komputer.

Pemindai dokumen fisik ke bentuk digital

Microphone

Microphone adalah peralatan input yang bertujuan untuk memasukkan/merekam input suara manusia secara langsung (voice recognizer).

Alat perekam suara untuk input audio

Peralatan Pemroses/CPU (Processing Device)

Motherboard

Motherboard adalah papan utama yang menampung komponen utama seperti mikroprosesor, RAM, ROM (dalam hal ini ROM BIOS atau sering disebut BIOS atau CMOS).

Papan induk yang menghubungkan semua komponen

Processor

Processor berfungsi sebagai CPU (Central Processing Unit), yaitu unit pengolah pusat yang merupakan otak dari suatu komputer yang mengatur kinerja komputer secara keseluruhan.

Otak komputer yang melakukan semua proses

Memory

Memory berfungsi sebagai unit penyimpanan. Memory dalam komputer terdiri dari dua yaitu memori internal dan memori eksternal.

Tempat penyimpanan data sementara

Peralatan Penyimpanan (Storage Device)

Hard Disk Drive (HDD)

Sebuah alat yang berfungsi untuk menyimpan data digital. HDD berjalan lebih lambat dibandingkan SSD, karena HDD melakukan proses baca/tulis pada kecepatan 0,1--1,7 MBps.

Penyimpanan data magnetik dengan kapasitas besar

Solid State Drive (SSD)

Sebuah alat yang berfungsi untuk menyimpan data digital. SSD berjalan lebih cepat dibandingkan HDD, karena SSD melakukan proses baca/tulis pada kecepatan 50--250 MBps.

Penyimpanan flash dengan kecepatan tinggi

Flashdisk

Flashdisk adalah alat penyimpanan data eksternal yang berfungsi untuk menyimpan dan memindahkan data digital dari satu komputer ke komputer lainnya.

Penyimpanan portabel dengan konektor USB

Peralatan Keluaran (Output Device)

Monitor

Alat untuk menampilkan hasil pemrosesan data.

Layar untuk menampilkan output visual

Printer

Peralatan output yang berfungsi untuk mencetak data pada kertas.

Alat pencetak dokumen ke media kertas

Speaker

Alat untuk menghasilkan suara. Tugas utamanya adalah mengubah data digital di dalam handphone, laptop atau flashdisk menjadi bunyi yang bisa kita dengar, seperti musik, suara guru di video pembelajaran, atau suara notifikasi.

Alat keluaran untuk suara dan audio

Projector

Alat untuk menampilkan gambar, video, atau teks dari komputer ke permukaan datar yang luas, seperti layar putih atau dinding. Jika layar laptop atau HP terasa terlalu kecil untuk dilihat banyak orang, maka Projector adalah solusinya. Projector ini sering digunakan untuk presentasi.

Proyektor untuk presentasi dan display besar

2. Perangkat Lunak (Software)

Software komputer adalah istilah umum yang digunakan untuk menggambarkan koleksi program, prosedur dan dokumentasi komputer yang melakukan beberapa tugas pada sebuah sistem komputer. Software terdiri atas tiga kategori, yaitu: sistem operasi, perangkat lunak aplikasi, dan perangkat lunak pemrograman. Software berisi perintah atau instruksi untuk menjalankan komputer.
